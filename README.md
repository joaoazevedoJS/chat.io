# chat.io

Chat in real time with socket.io

websie: <https://chatio.joaoazevedojs.com.br/>

You'll need [Node.js](https://nodejs.org) installed on your computer!

[![Chat.io](./Home.PNG)](https://chatio.joaoazevedojs.com.br/)

## Backend

```bash
  cd backend
  yarn
  yarn dev
```

> or use npm

```bash
  cd backend
  npm install
  npm run dev
```

### Frontend

```bash
  cd frontend
  yarn
  yarn start
```

> or use npm

```bash
  cd frontend
  npm install
  npm start
```
